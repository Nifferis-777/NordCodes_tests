# Задание: покрыть тестами java-приложение, представляющее собой классический веб-сервис на Spring Boot. 

## Запуск приложения осуществляется следующей командой:

```

java -jar -Dsecret=qazWSXedc -Dmock=http://localhost:8888/ internal-0.0.1-SNAPSHOT.jar 
 
```

### Используемый стек технологий:
1) ` Java `

2) ` Maven `

3) ` JUnit5 ` 

4) ` WireMock `

5) ` RestAssured `

6) ` Allure `

## Формирование Allure-отчета после прогона тестов и его просмотр

Для того чтобы сформировать Allure-отчет после прогона тестов, необходимо предварительно установить ` Allure ` , а затем выполнить в консоли IDE следующую команду:

```

allure serve

```
В открывшемся отчете перейти в нужную вкладку, например "Behaviors". В ней отображаются подробные "читаемые" отчеты о прохождении тестов.
